PROJECT TITLE:Driving For Professionals
PURPOSE OF PROJECT: Enseñar por medio de unos manuales divididos
por 5 secciones (conocimientos del vehiculo, señales preventivas,
señales reglamentarias, señales informativas, otras señales).
y evaluar a los usuarios con distintas pruebas teoricas, 
respondiendo unos cuestionaros por cada una de las secciones  y 
practicas con unos test de manejo como simulador de manejo
con distintas situaciones enseñada en los manuales de cada
seccion.
VERSION or DATE:17/09/2020 
HOW TO START THIS PROJECT:
Greenfoot, github
Para hacer funcion de la aplicacion se puede con el
ejecutable .jar por medio de java o la aplicaion de greenfoot
AUTHORS: Cristian Guillermo Serrano & Jean Carlos Portilla
USER INSTRUCTIONS: 
Comenzamos con el menu de inicio con las distintas opcioines que son las pruebas,
el manual de teoria para las pruebas teoricas, las instrucciones de controles para 
las pruebas practicas y el boton de salir.
al acceder al manual empiezara a mostrarse por orden de secciones para pasar pagina 
usa las flechas indicadas, para volver al menu debe clickear el boton a nterior en
la primera pagina o la derecha en la ultima pagina.
al accerder al menu de pruebas se mostrara cda una de las secciones con cada una de
sus respectivas pruebas, al clickear cada una de las secciones lo mandara al respectivo
test teorico que debe responder todas las preguntas correctamente para acceder a la
prueba practica y ya en la prueba practica debe hacer lo indicado en el mapa, al 
finalizar la prueba se mandara al usuario al menu para que decida que otra prueba
quiere hacer.
